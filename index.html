Steve's Notes — How to Post a Game
  1. Collect raw game stats (box score, play-by-play, etc.)
  2. Paste the data into Claude Code on your phone to generate the HTML report file
  3. Upload the HTML file to the games/ folder on GitHub
  4. Update games.json with a new entry (title, date, time, venue, score, filename)
  5. Commit — Vercel auto-deploys and this page picks up the new game
